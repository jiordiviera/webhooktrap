# Installation

Choose your deployment method:

## Local Development (Docker)

### Prerequisites
- Docker & Docker Compose
- 2GB RAM
- Git

### Steps

1. **Clone the repository**
```bash
git clone https://github.com/jiordiviera/webhooktrap.git
cd webhooktrap
```

2. **Copy environment template**
```bash
cp .env.example .env
```

3. **Start services**
```bash
docker compose up
```

4. **Access the app**
- Web: http://localhost:7777
- API: http://localhost:3333

The database initializes automatically on first run.

### Stop & Cleanup

```bash
# Stop services
docker compose down

# Stop and remove database
docker compose down -v
```

For production, choose one of:

Auto-deploys from GitHub, handles SSL/domains.

Manages deployments across regions with edge caching.

## Verify Installation

### Check Web
```bash
curl http://localhost:7777
```
Should return HTML (Next.js landing page).

### Check API
```bash
curl http://localhost:3333/api/v1/inboxes
```
Should return JSON (requires authentication).
